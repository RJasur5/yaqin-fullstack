# Findix Project

This is the root directory of the Findix project, containing both the backend and frontend.

## Quick Start

To launch both the backend and frontend with a single command, use one of the following:

### Windows Batch (e.g. from Explorer or cmd)
- Double-click `run_all.bat`
- Or run `run_all.bat` in Command Prompt.

### PowerShell
- Run:
  ```powershell
  .\run_all.ps1
  ```

## Structure
- `backend/`: FastAPI backend service.
- `findix_app/`: Flutter mobile and web application.
- `uploads/`: Directory for uploaded files.
