# Инструкция по переносу проекта Yaqin на Mac

Этот файл поможет вам быстро запустить приложение на вашем Mac и установить его на iPhone.

## 1. Подготовка Mac
Если вы еще не настраивали Flutter на Mac:
1.  **Xcode**: Установите из App Store. После установки запустите его и разрешите установку дополнительных компонентов.
2.  **Flutter SDK**: Скачайте с [flutter.dev](https://flutter.dev/docs/get-started/install/macos).
3.  **CocoaPods**: Откройте терминал и введите:
    ```bash
    sudo gem install cocoapods
    ```

## 2. Развертывание проекта
1.  Распакуйте полученный ZIP-архив в удобную папку.
2.  Откройте терминал в папке проекта (`yaqin_project`) и выполните:
    ```bash
    cd findix_app
    flutter pub get
    cd ios
    pod install
    ```

## 3. Настройка подписи (Signing)
Так как у вас есть **платный аккаунт**, всё будет работать официально:
1.  Откройте файл `ios/Runner.xcworkspace` в Xcode.
2.  В левой колонке выберите самый верхний пункт **Runner**.
3.  Перейдите на вкладку **Signing & Capabilities**.
4.  В поле **Team** выберите свой аккаунт Apple (если его нет, нажмите "Add Account").
5.  Xcode автоматически создаст сертификаты (Provisioning Profiles).

## 4. Запуск на iPhone
1.  Подключите iPhone кабелем к Mac.
2.  В верхней части Xcode выберите ваш iPhone в списке устройств.
3.  Нажмите кнопку **Play** (треугольник) или выполните в терминале:
    ```bash
    flutter run
    ```

## 5. Подключение к серверу (Backend)
Если вы запускаете сервер (Backend) на Windows, а приложение на iPhone:
- Убедитесь, что оба устройства в одной Wi-Fi сети.
- В приложении (`lib/config/api_config.dart`) должен быть указан IP-адрес вашего Windows-компьютера.
- Если вы используете **Ngrok** или **LocalTunnel**, просто вставьте ссылку в конфиг.
